Guide to admin panel

1. Adding to the database.
If you want to add a new event to the database, you need to go in "Database" section, in the tab "Add event". After that, complete all the textboxes, but be very carefull, because when you see "please check ..." it means that you need to search in the "Informations about data" section the right ID for your situation.

2. Update an event from your database.
If you want to update an event from your database, you need to know its ID.

3. Delete an event from database.
If you need to delete an event, you need to know its ID.
